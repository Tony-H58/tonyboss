## 1️⃣ 驗布標準概述

聚陽採用 **MK標準** 和 **GU標準** 兩套驗布系統：
- **MK標準**：四分制，主要客戶應用
- **GU標準**：十點制，美標客戶應用

### 驗布流程
```
驗布開始
    ↓
按規定抽取樣品（最少1500碼）
    ↓
進行單疋（Roll）檢驗
    ↓
記錄檢驗結果
    ↓
計算平均值
    ↓
進行單色（Lot）判級
    ↓
確定出貨結論
    ↓
填寫報告
    ↓
驗布結束
```

## 2️⃣ MK 標準 vs GU 標準

| 項目 | MK 標準 | GU 標準 |
|------|--------|--------|
| **扣點制度** | 四分制 | 十點制 |
| **單疋判級** | A/B/C 三級 | Pass/Discuss/Fail 三級 |
| **客戶適用** | 一般客戶 | 美標客戶 |
| **檢驗項目** | 12-15項 | 15-20項 |
| **扣點標準** | ≦10(A), 10.1~33(B), >33(C) | 見十點制表 |
| **使用報告** | QF-001C | QF-001G |

## 3️⃣ MK 標準詳解

### 3.1 四分制扣點系統

| 等級 | 扣點 | 含義 |
|------|------|------|
| **1級** | 1分 | 微弱瑕疵，遠處難以察覺 |
| **2級** | 2分 | 輕微瑕疵，近處方可察覺 |
| **3級** | 3分 | 中度瑕疵，明顯可見 |
| **4級** | 4分 | 嚴重瑕疵，非常明顯，影響外觀或功能 |

### 3.2 MK 單疋判級標準

**檢驗項目與評級標準**

#### A. 布面瑕疵（最重要）⭐⭐⭐
| 等級 | 標準 | 定義 |
|------|------|------|
| **A 級** | ≦ 10 分/100碼 | 瑕疵少，品質優良 |
| **B 級** | 10.1 ~ 33 分/100碼 | 瑕疵中等，需討論是否接受 |
| **C 級** | > 33 分/100碼 | 瑕疵多，不合格 |

**計算方法**：
```
平均扣點 = 總扣點數 ÷ 驗布長度(百碼) × 100
扣點比例 = 平均扣點數 × 0.3 ÷ 100
補布比例 = 扣點比例 - 3%

示例：驗布300碼，扣點20分
平均扣點 = 20 ÷ 3 = 6.67 分 → A級
扣點比例 = 6.67 × 0.3 ÷ 100 = 2%
補布比例 = 2% - 3% = -1%（無補布）
```

#### B. 布面外觀
| 等級 | 標準 |
|------|------|
| **A 級** | 比對標樣：輕微差異 |
| **B 級** | 無此級 |
| **C 級** | 比對標樣：明顯差異 |

#### C. 手感
| 等級 | 標準 |
|------|------|
| **A 級** | 比對標樣：輕微差異 |
| **B 級** | 無此級 |
| **C 級** | 比對標樣：明顯差異 |

#### D. 歪斜（緯斜/緯弧）⭐
| 等級 | 標準 | 計算 |
|------|------|------|
| **A 級** | ≦ 3% | 長度差÷布寬×100 |
| **B 級** | 無此級 | |
| **C 級** | > 3% | |

**常見歪斜圖形**：
```
正常        緯斜右      緯斜左      緯弧
[====]    [====/]    [/====]    [~~~~~]
```

#### E. 色差 - 邊中差/頭尾差/疋差
| 等級 | 標準 | 參考 |
|------|------|------|
| **A 級** | 輕微差異 | 對色儀≧4.5級 |
| **B 級** | 無此級 | |
| **C 級** | 明顯差異 | 對色儀<4.5級 |

#### F. 色差 - 缸差（Lot間）
| 等級 | 標準 | 參考 |
|------|------|------|
| **A 級** | 輕微差異 | 對色儀≧4級 |
| **B 級** | 只提供數據，不評級 | 說明差異程度 |
| **C 級** | 明顯差異 | 對色儀<4級 |

#### G. 規格 - 有效幅寬
| 等級 | 標準 |
|------|------|
| **A 級** | 0" ~ +2" |
| **B 級** | 無此級 |
| **C 級** | < 0" 或 嚴重差異 |

#### H. 規格 - 克重(g/m²)
| 等級 | 標準 |
|------|------|
| **A 級** | ≦ 5% 差異 |
| **B 級** | 無此級 |
| **C 級** | > 5% 差異 |

#### I. 規格 - 條距/格距循環
- 每 20" 測量一次
- 只提供數據，不評級

#### J. 異味
| 等級 | 標準 |
|------|------|
| **A 級** | 無任何異味 |
| **B 級** | 無此級 |
| **C 級** | 有明顯異味 |

### 3.3 MK 單色（Lot）結論判級

**布面瑕疵結論**

| 等級 | 條件 | 定義 |
|------|------|------|
| **A 級** | ≦ 10 分/100碼<br/>且瑕疵比例 ≦ 3% | 優良品，可直接出貨 |
| **B 級** | 10.1 ~ 33 分/100碼<br/>且瑕疵比例 3.1% ~ 10% | 待討論，需評估 |
| **C 級** | > 33 分/100碼<br/>且瑕疵比例 > 10% | 不合格，需返工或報廢 |

**C級比例判定**
```
C級缸結論：
- C級比例 ≦ 10% → A 級（整缸接受）
- C級比例 > 10% → C 級（整缸不接受）
```

## 4️⃣ MK 出貨結論規定

### 整缸判級規則

| 情況 | 結論 | 備註 |
|------|------|------|
| **全A級** | ✅ A 級（可直接出貨） | 最佳 |
| **A + 少量B** | ✅ A 級 | B級<10% |
| **多量B或有C** | ⚠️ B 級（待討論） | 需業務確認 |
| **大量C** | ❌ C 級（報廢/返工） | C級>10% |

### 部分打下規定

**打下類型**

| 類型 | 說明 | 結論判定 |
|------|------|--------|
| **單疋打下** | 只打下 1 匹（Roll） | 該匹為C級<br/>其餘匹保持原級 |
| **單缸打下** | 只打下 1 缸（Lot） | 該缸為C級<br/>其餘缸保持原級 |
| **部分打下** | 打下部分匹或缸 | 複雜判定<br/>見詳細說明 |

## 5️⃣ GU 標準詳解

### 5.1 十點制扣點系統

GU 美標採用詳細的十點制扣分：

| 缺陷嚴重度 | 扣點 | 定義 |
|-----------|------|------|
| **極微** | 0.1~0.5 | 肉眼難見 |
| **輕微** | 1~2 | 近距離方見 |
| **中等** | 3~4 | 明顯可見 |
| **嚴重** | 5~7 | 非常明顯 |
| **致命** | 8~10 | 完全不合格 |

### 5.2 GU 單疋判級標準

**扣點標準（Per 100 linear-yards）**

| 等級 | Pass | Discuss | Fail |
|------|------|---------|------|
| **範圍** | ≦ 10 pts | — | > 10 pts |

**檢驗項目表**（與MK相似但評分更詳細）
- 布面瑕疵
- 外觀外觀
- 手感
- 歪斜
- 色差（三個維度）
- 規格（幅寬、克重、循環）
- 異味

### 5.3 GU 單色（Lot）結論

| 等級 | 條件 |
|------|------|
| **Pass** | 全部或大部分為 Pass |
| **Discuss** | 有 Discuss 需討論 |
| **Fail** | 有 Fail 不合格 |

## 6️⃣ 檢驗重點與技巧

### 檢驗項目優先級

🔴 **最重要（必檢）**
1. 布面瑕疵（占評級 80%）
2. 色澤（對客戶感受最大）
3. 歪斜（影響剪裁成本）

⚠️ **重要（詳檢）**
4. 克重/規格
5. 手感
6. 外觀

✅ **參考（記錄）**
7. 異味
8. 循環尺寸

### 快速判級技巧

```
看扣點數
    ↓
< 10 分 → A 級 ✓
10~33 分 → B 級 ⚠️
> 33 分 → C 級 ✗

再看色差和歪斜
    ↓
色差明顯 / 歪斜 > 3% → 可能降級至 B/C

最後綜合判定
    ↓
若無其他明顯缺陷 → 按扣點判級
若有色澤/歪斜問題 → 可能需降級或討論
```

## 7️⃣ 聚陽品管部門規定

### 工作規範（2020）

#### 驗貨量與時間
- **最小驗貨量**：1500碼/次
- **布量少**：提高抽驗比率
- **報告時間**：當天完成驗貨後 2小時內
- **工作紀錄**：記錄在報告上

#### 工作時間安排
- **常規時間**：需完成安排驗貨數量
- **假日/晚上**：依實際任務需求配合加班
- **提早離開**：16:00前離開廠區需主管同意
- **加班審批**：21:00後加班需總經理核准

#### 出差費用標準（月度申報）

| 費用項目 | 標準 | 備註 |
|---------|------|------|
| **住宿費** | 160元/晚 | 酒店住宿；超出自付 |
| **備選住宿** | 100元/晚 | 廠宿或回住處 |
| **誤餐費** | 30元/天 | 執行任務每天固定 |
| **驗貨獎勵** | 30元/次 | 1家供應商1500碼以上；可累加 |
| **長途工作** | 20元/次 | 火車>2h或高鐵>3h |
| **手機費補貼** | 250元/月 | 品管人員（在職期間） |
| **高溫補貼** | 5元/天 | 6/15~9/15全天出差 |
| **夜間加班** | 30元/次 | 20:00後加班（需主管同意） |

#### 交通工具規定
**優先級順序**：
1. 巴士/火車/動車/高鐵（二等座）
2. 市內：公交/地鐵/廠商接送
3. 出租車：需事先報備，機打發票

#### 請假規定

| 類型 | 提前申請 | 備註 |
|------|--------|------|
| **計畫性**（2-4天） | 2週前 | 連續假期 |
| **計畫性**（≥5天） | 1個月前 | 長期假期 |
| **臨時性** | 當天電話通知 | 2天內完成手續 |

#### 行為準則（必須遵守）⭐⭐⭐

🚫 **禁止行為**
1. 不得接受供應商金錢馈贈
2. 不得接受工作餐以外的招待
3. 不得擅自更改行程
4. 不得危害公司利益或名譽
5. 不得虛報費用

✅ **必須行為**
1. 誠實報銷所有費用
2. 遵守公司規定
3. 配合公司安排驗貨任務
4. 及時提交驗貨報告
5. 小心保管平板設備

## 8️⃣ 特殊情況判定

### 部分打下（Partial Reject）

**何時使用部分打下**：
- 整缸大部分A級，少數疋為C級
- 整缸大部分A級，1缸為C級
- 複雜情況需評估

**判定方式**：
```
單疋打下（單Roll）
├─ 該疋判為C級
├─ 其他疋保持原級
└─ 整缸級別重新計算

單缸打下（單Lot）
├─ 該缸判為C級
├─ 其他缸保持原級
└─ 整個訂單級別重新計算

部分打下（Lot&Roll混合）
├─ 複雜計算
├─ 需業務和品管確認
└─ 需詳細說明理由
```

### 驗貨空跑（Inspection Missed）

**定義**：驗貨人員到達但工廠無布料檢驗

**處理**：
- 記錄空跑事由
- 按規定報銷費用（不報驗貨獎勵）
- 保留記錄以備查詢

### 中段驗貨（In-Line Inspection）

**定義**：生產過程中的中間檢驗

**特點**：
- 反應生產問題
- 可能影響最終判級
- 需與成品驗貨結果結合評估

## 9️⃣ 驗布報告填寫

### MK 版本（QF-001C）
**必填項目**：
- 布料編號/客戶
- 驗布日期/人員
- 驗布長度/位置
- 各項檢驗評分
- 單疋判級結果
- 整缸結論
- 簽名確認

### GU 版本（QF-001G）
**必填項目**：
- （同上，但採用十點制評分）

### 填寫要點
✅ **規範要求**
1. 用黑色或藍色筆
2. 字跡清晰易辨
3. 數字準確無誤
4. 簽名須完整
5. 日期格式統一

❌ **常見錯誤**
1. 扣點計算錯誤
2. 判級邏輯不符
3. 簽名不清
4. 缺少必填項
