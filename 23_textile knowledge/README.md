## 🎯 按工作場景快速查詢

### 早上上班第一件事
1. **查看昨天的缺陷統計** → `品管檢驗清單.md` → 📊 缺陷統計表
2. **了解今天重點** → 與主管溝通
3. **準備檢驗工具** → 燈箱、尺子、pH試紙、色差儀

### 紗源驗收時
📋 **使用清單**：`品管檢驗清單.md` → 🧵 紗源驗收清單
- [ ] 檢查外觀顏色
- [ ] 檢查規格（號數、重量）
- [ ] 品質檢驗（條幹、異纖、死棉）
- [ ] 感觸檢驗（強力、手感）

**遇到不確定**：查看 `瑕疵識別速查表.md` → 🧵 紗線因素

### 織造過程檢驗
📋 **使用清單**：`品管檢驗清單.md` → 📋 織造過程檢驗清單
- [ ] 織針磨損
- [ ] 針路檢查
- [ ] 油污檢查
- [ ] 張力檢查

**發現問題**：查看 `瑕疵識別速查表.md` → 🏭 織造因素

### 染色過程檢驗
📋 **使用清單**：`品管檢驗清單.md` → 📋 染色過程檢驗清單
- [ ] 溫度控制
- [ ] 色澤均勻
- [ ] 泳移檢查
- [ ] pH值檢驗

**發現色斑/色花等**：查看 `染色疵點診斷與解決方案.md`
- 色斑 → 第3項
- 色花 → 第7項
- 泳移 → 第2項

### 定型與整理檢驗
📋 **使用清單**：`品管檢驗清單.md` → 📋 定型與整理檢驗清單
- [ ] 溫度、時間
- [ ] 毛高均勻
- [ ] 無跳刀痕
- [ ] 無陰陽色

### 成品品檢
📋 **使用清單**：`品管檢驗清單.md` → 成品品檢清單
1. **外觀檢驗**
   - 用燈箱檢查顏色（色差儀對比）
   - 逆光查看瑕疵
   - 測量尺寸和重量

2. **缺陷判定**
   - 查看 `瑕疵識別速查表.md` 確認缺陷類型
   - 使用 `染色疵點診斷與解決方案.md` 判定原因
   - 決定：優等品/一等品/二等品/報廢

3. **記錄**
   - 填寫 `品管檢驗清單.md` → 📝 每日檢驗記錄表

### 遇到問題時
1. **想不起來這是什麼缺陷**
   → `瑕疵識別速查表.md` → 按來源（紗/織/染/加工）查找

2. **不知道如何改善**
   → `染色疵點診斷與解決方案.md` → 13大疵點診斷表

3. **不知道是否合格**
   → `品管檢驗清單.md` → 快速決策樹

4. **不知道術語是什麼意思**
   → `紡織專業術語速查表.md` → 按分類快速查詢

## 📖 文件詳細說明

### 1️⃣ 瑕疵識別速查表
**文件**：`01_瑕疵識別/瑕疵識別速查表.md`
**用途**：識別面料缺陷，了解來源
**內容**：
- ✅ 紗線因素 7種（條幹、死棉、異纖等）
- ✅ 織造因素 5種（針路、油污、毛羽等）
- ✅ 染整因素 7種（折痕、色斑、黃斑等）
- ✅ 加工因素 5種（搖粒絨缺陷為例）
- ✅ 快速決策表

**何時使用**：
- 看到陌生的布面瑕疵時 ⭐
- 要確認缺陷嚴重度時 ⭐
- 要向上級報告時

**快速查找技巧**：
```
發現缺陷
    ↓
看它的形狀：
├─ 直向條紋 → 查 🏭 織造因素 → 針路或油污
├─ 橫向條紋 → 查 🏭 織造因素 → 橫條
├─ 色斑/色點 → 查 🎨 染整因素 → 色斑
├─ 折痕 → 查 🎨 染整因素 → 折痕
└─ 小棉粒 → 查 🧵 紗線因素 → 死棉
```

**何時使用**：
- 發現染色問題，要決定如何改善時 ⭐
- 染色缸差，要找原因時 ⭐
- 要調試工藝參數時 ⭐

**常用查詢**：
```

→ 查：是浸染還是軋染？

問題3：缸差
→ 第6項：缸差
→ 查：根本原因分析表
```

```
8:00 - 上班
│
├─ 查看昨天缺陷統計 (品管檢驗清單 → 📊 表)

└─ 準備檢驗工具

9:00-12:00 - 紗源/織造檢驗
│
├─ 使用 品管檢驗清單 → 紗源/織造清單
├─ 發現問題 → 查 瑕疵識別速查表
└─ 記錄異常 → 每日記錄表

12:00-14:00 - 午餐 + 回顧記錄

14:00-17:00 - 染色/整理檢驗
│
├─ 使用 品管檢驗清單 → 染色/整理清單
├─ 發現問題 → 查 染色疵點診斷方案
├─ 與工廠溝通改善
└─ 記錄 → 每日記錄表

17:00-18:00 - 成品檢驗
│
├─ 使用 品管檢驗清單 → 成品品檢清單
├─ 外觀檢驗（燈箱、色差儀）
├─ 缺陷判定（查 瑕疵速查表 + 術語表）
├─ 分等級（優等/一等/二等/報廢）
└─ 記錄 → 每日記錄表

18:00 - 下班
│
└─ 整理記錄，更新統計表
```

### 常見場景應對

**場景1：看到一個奇怪的紋路，不知道是什麼**
1. 打開 `瑕疵識別速查表.md`
2. 按形狀判斷（直向/橫向/點狀）
3. 找到對應缺陷
4. 查看「原因」和「預防」

**場景2：工廠說要調整工藝，問意見**
1. 打開 `染色疵點診斷與解決方案.md`
2. 找到相關疵點
3. 查看「改善措施」
4. 建議最優方案

**場景3：客戶來抽檢，要確認合格標準**
1. 打開 `品管檢驗清單.md`
2. 跳到「瑕疵判定標準」
3. 按照標準逐項檢查
4. 決定等級並簽名記錄

**場景4：要向主管報告品質情況**
1. 打開 `品管檢驗清單.md`
2. 填寫「📊 缺陷統計表」
3. 分析趨勢（↑/→/↓）
4. 提出改善措施建議
