# 品管檢驗清單與標準

## 📋 日常檢驗清單

### 紗源驗收清單
**檢驗時機**：布料到達品管部門時

- [ ] **外觀檢驗**
  - [ ] 紗色是否正確（與訂單對應）
  - [ ] 紗線號數是否符合（檢查標籤）
  - [ ] 筒子（卷軸）是否完好
  
- [ ] **規格檢驗**
  - [ ] 重量是否符合（隨機抽檢）
  - [ ] 紗線號數確認（10支、20支等）
  
- [ ] **品質檢驗**
  - [ ] 條幹不均：目視逆光查看 ✓ / ✗
  - [ ] 死棉：特別注意淺色布 ✓ / ✗
  - [ ] 異纖：逆光查看是否有異色點 ✓ / ✗
  - [ ] 汙紗：是否有污漬 ✓ / ✗
  - [ ] 粗節：逆光查看 ✓ / ✗
  
- [ ] **感觸檢驗**
  - [ ] 紗線強力：輕拉是否易斷 ✓ / ✗
  - [ ] 手感：是否粗硬 ✓ / ✗
  
- [ ] **結論**
  - [ ] 合格 ✓ 可入庫
  - [ ] 條件接受 ⚠️ 記錄備註
  - [ ] 不合格 ✗ 退回供應商

---

### 織造過程檢驗清單
**檢驗時機**：織造進行中，每2小時檢查一次

| 檢驗項目 | 標準 | 檢查方法 | 狀態 |
|---------|------|---------|------|
| **織針磨損** | 無明顯磨損 | 目視檢查織針 | ✓/⚠️/✗ |
| **針路** | 無直向條紋 | 逆光觀察 | ✓/⚠️/✗ |
| **油污** | 無油污痕跡 | 目視 + 觸摸 | ✓/⚠️/✗ |
| **飛花** | 無外來雜質 | 目視檢查 | ✓/⚠️/✗ |
| **張力** | 均勻一致 | 檢查張力表 | ✓/⚠️/✗ |
| **毛羽** | 合理範圍內 | 手摸檢查 | ✓/⚠️/✗ |
| **生產速度** | 符合工藝 | 檢查機器表 | ✓/⚠️/✗ |
| **機器溫度** | 正常範圍 | 溫度計檢查 | ✓/⚠️/✗ |

**發現問題時**：立即記錄，通知操作人員改正

### 染色過程檢驗清單
**檢驗時機**：
- 升溫時：每10°C檢查一次
- 保溫時：每15分鐘檢查一次
- 降溫時：監控溫度

| 檢驗項目 | 標準 | 檢驗方法 | 簽名 |
|---------|------|---------|------|
| **水溫** | ±2°C 以內 | 溫度表讀數 | ___ |
| **色澤均勻** | 無深淺差 | 目視布面 | ___ |
| **泳移** | 正反面色一致 | 翻面觀察 | ___ |
| **色斑** | 無色點/色塊 | 逆光查看 | ___ |
| **折痕** | 無折痕 | 展開布面檢查 | ___ |
| **液位** | 符合容量 | 檢查液面 | ___ |
| **迴圈狀態** | 流暢無梗阻 | 檢查進出液 | ___ |
| **pH值**（最終） | 6.5-7.5 | pH試紙測試 | ___ |

**重點溫度節點檢驗表**

```
升溫過程：
├─ 25°C：加消泡劑
├─ 40°C：加鹽（活性）或化解分散染料
├─ 60°C：（活性最高溫）加鹼，或繼續升溫
├─ 90°C：檢查色澤
├─ 130°C：（分散最高溫）保溫
└─ 降溫至50°C：水洗
```

## 成品品檢清單

### 外觀檢驗
**工具**：燈箱、尺子、色差儀

| 檢驗項 | 標準 | 合格判定 |
|-------|------|--------|
| **顏色** | 與標樣一致 | ΔE ≤ 1.0（色差儀） |
| **外觀瑕疵** | 詳見瑕疵表 | 見下方詳表 |
| **尺寸** | ±5% 以內 | 用尺測量 |
| **重量** | ±5% 以內 | 用秤稱量 |
| **幅寬** | 允差±2cm | 用尺測量 |

### 瑕疵判定標準

**優等品**：無瑕疵 ✓

**一等品**（允許輕微瑕疵）：
- [ ] 瑕疵面積 < 2cm²，且數量 ≤ 2個
- [ ] 無色澤、強力等功能性瑕疵
- [ ] 符合客戶規格

**二等品**（允許明顯瑕疵但不影響使用）：
- [ ] 瑕疵面積 < 5cm²
- [ ] 瑕疵不在可見面
- [ ] 客戶同意

**不合格品**（報廢或降級）：
- [ ] 瑕疵面積 ≥ 5cm²
- [ ] 色澤、強力不符合
- [ ] 客戶無法接受

## 🎯 品檢工作流程

```
原料入庫
    ↓
紗源驗收 ← 不合格：記錄+退回
    ↓ ✓ 合格
織造過程檢驗 ← 發現問題：通知改正
    ↓
漂染過程檢驗 ← 發現問題：停機調試
    ↓
定型整理檢驗 ← 不合格：返工
    ↓
成品檢驗 ← 發現瑕疵：分級
    ↓
    ├─ 優等品 → 直接出貨
    ├─ 一等品 → 正常出貨
    ├─ 二等品 → 降級出貨
    └─ 不合格 → 報廢/返工
```
