## 🧵 纖維材料術語（15個）

### 天然纖維
- **棉**：Cotton。吸濕性好，舒適，易皺。常用於T恤、內衣
- **麻**：Linen。強力高，涼爽，易皺。常用於休閒裝
- **毛**：Wool。保暖，蓬鬆，易縮水。常用於冬衣
- **絲**：Silk。光澤，柔軟，昂貴。常用於高檔衣料

### 化學纖維
- **聚酯（滌綸）**：Polyester/PET。強力高，易打理，吸濕差。最常用
- **聚酯長絲**：PET Filament。連續長絲，FDY/POY/DTY等
  - **FDY**：Fully Draw Yarn 全延伸絲。已定向定型，可直接使用
  - **POY**：Pre-Oriented Yarn 預取向絲。需後加工，製成DTY
  - **DTY**：Draw Texture Yarn 假撚彈性紗。經假撚加工，具彈性
  - **ITY**：Intermingled Texture Yarn 噴節紗（氣紡紗）
- **尼龍（聚醯胺）**：Nylon/PA。強力好，耐磨，貴。常用於運動裝、褲子
- **晴綸（聚丙烯腈）**：Acrylic。手感像羊毛，保暖。常混紡
- **粘膠**：Viscose/Rayon。吸濕好，光澤好，易皺。常用於針織

### 特殊纖維
- **超細纖維**：Microfiber。每根小於0.5d。觸感柔軟，吸濕快
- **海島紗**：Sea-island fiber。每根可達0.1d。超細、高級
- **中空纖維**：Hollow fiber。中空結構，保暖輕。常用於填充
- **異型斷面**：Shaped filament。十字、三角形等。特殊光澤

## 🧶 紗線類型術語（12個）

### 短纖紗
- **環錠紡紗**：Ring spinning。品質最好，但速度慢
- **緊密紡**：Compact spinning。短纖改進型，毛羽少
- **賽絡紡（Siro紗）**：Siro spinning。細數短纖高品質選擇
- **氣流紡**：Air-jet spinning。速度快，成本低
- **渦流紡**：Vortex spinning。品質好，速度快
- **噴氣紡**：Pneumatic spinning。可短纖混長纖

### 長纖紗
- **假撚紗**：Draw-textured yarn（DTY）。有彈性，蓬鬆
- **包芯紗**：Covered yarn。短纖包住長纖，綜合優點
- **花式紗**：Fancy yarn。外觀有變化：圈圈紗、節點紗等

### 紗線規格
- **支數**：Count。如20支棉、100D滌綸
  - 棉紗：支數越高越細
  - 長纖：D（旦數）表示，D越小越細
- **纖度**：Denier/Den。纖維細度單位
  - 75D：細
  - 150D：中
  - 300D：粗

## 🧵 織物規格術語（10個）

### 基本規格
- **門幅/幅寬**：Width。布的寬度，常見54"、58"、60"
- **克重**：Weight。單位面積重量（g/m²）或碼重（g/yd²）
- **碼重**：Yard weight。每碼長的重量（g）
  - 計算：碼重 = 克重 × 門幅(m) × 0.02323
- **密度**：Density
  - **經密**：Warp count（WPI）。每英寸經向紗數
  - **緯密**：Weft count（CPI）。每英寸緯向紗數
  - **針密**（針織）：Gauge。每英寸針數，如18G（18針/英寸）

### 規格檢驗
- **縮水率**：Shrinkage。洗滌後尺寸變化
  - 經向縮水率 > 緯向 通常
  - 棉布：5-10%；滌綸：< 2%
- **斜度**（針織）：Bias angle。緯編布的斜向角度
  - 正常範圍：15-25°
  - 過大：布易滑，寬鬆
  - 過小：布易拱起，緊身

## 🏭 工藝術語（20個）

### 紡紗工序
- **配棉**：Blending。混合不同棉花品種
- **梳棉**：Carding。纖維平行整理
- **精梳**：Combing。進一步提高品質
- **粗紗**：Sliver。尚未有捻度的纖維束
- **粗紗機**：Drawing frame。拉伸短纖
- **紡錠**：Spindle。旋轉加捻的部件
- **加捻**：Twisting。給紗線加捻度

### 織造工序（梭織）
- **經線/織經**：Warp。縱向
- **緯線/織緯**：Weft。橫向
- **打緯**：Beat。每次打進一根緯線
- **織速**：Loom speed。rpm（轉數）
- **張力**：Tension。紗線拉力

### 織造工序（針織）
- **線圈**：Loop。針織基本單位
- **圈高**：Loop height。線圈高度
- **成圈**：Loop formation。針織過程
- **脫圈**：Dropped stitch。線圈脫落

### 染整工序
- **前處理**：Pre-treatment。燒毛、煮布、漂白
- **定型**：Heat setting。用熱固定布的尺寸和風格
- **軋光**：Calendering。用熱壓使布光亮
- **柔軟**：Softening。加柔軟劑

## 🎨 缺陷術語（25個）

### 紗線缺陷
- **條幹不均**：Unevenness of yarn。粗細不一
- **死棉**：Dead cotton。無法上色的纖維
- **異纖**：Impurities/Foreign fibers。異色纖維
- **汙紗**：Stained yarn。有污漬
- **粗節**：Nep/Slub（<1寸）。片段偏粗
- **粗紗**：Slub（>1寸）。連續偏粗

### 織造缺陷
- **針路**：Needle line。織針磨損痕跡
- **油污**：Oil stain。油漬痕跡
- **飛花**：Flying fibers。外來雜質
- **橫條**：Horizontal streak。橫向條紋
- **毛羽**：Fuzz/Nap。纖維豎起
- **斜度異常**：Bias angle abnormal。針織布斜度不對

### 染色缺陷
- **泳移**：Color bleeding/Migration。色素游離
- **色斑/色點**：Color spot。深色點狀
- **色花**：Color unevenness。上色不均
- **色條**：Color stripe。條狀色差
- **缸差**：Dye lot difference。不同缸顏色不一致
- **白芯**：White core。紗線中心顏色淡
- **色淺**：Pale color。色澤偏淺
- **色光不對**：Wrong color tone。偏紅/藍/黃/綠

### 整理缺陷
- **折痕/死痕**：Fold mark。無法消除的痕跡
- **水折痕**：Water mark。脫水形成的折痕
- **刮痕**：Abrasion mark。機械刮傷
- **油漬**：Oil spot。軟油未均勻分散
- **黃斑**：Yellow stain。放置時間過長形成
- **跳刀痕**：Skip mark。機器突然停機
- **陰陽色**：Uneven dyeing on edges。布邊兩側色差

## 📊 質量指標術語（15個）

### 物理性能
- **強力**：Strength。拉斷所需的力
  - **斷裂強力**：Breaking strength（N）
  - **斷裂伸長**：Breaking elongation（%）
- **撕裂強力**：Tear strength。撕裂所需的力
- **耐磨性**：Abrasion resistance。磨損抵抗力
  - **耐摩擦牢度**：Rubbing fastness（1-5級，5最好）
- **硬度**：Stiffness。布的硬軟程度
  - 軟：舒適
  - 硬：挺立
- **伸縮性**：Elasticity。拉伸後恢復能力
  - **彈性恢復率**：Recovery after stretching

### 化學性能
- **pH值**：pH value。酸鹼度
  - 標準：6.5-7.5
  - < 6.5：過酸
  - > 7.5：過鹼
- **色牢度**：Colorfastness
  - **日曬牢度**：Light fastness（1-8級）
  - **洗滌牢度**：Wash fastness（1-5級）
  - **摩擦牢度**：Rubbing fastness（干/濕，1-5級）
  - **昇華牢度**：Sublimation fastness（涤綸重要）
- **縮水率**：Shrinkage
  - **熱縮**：Heat shrinkage。定型時的收縮
  - **濕縮**：Wet shrinkage。水洗時的收縮

### 衛生性能
- **甲醛含量**：Formaldehyde content。標準：<75ppm
- **重金屬**：Heavy metal content。如鉛、鎘、鉻等
- **pH值**：見上文

## 快速對照表

### 缺陷嚴重度評級

| 等級 | 判定 | 處理方案 |
|------|------|---------|
| 🔴 致命缺陷 | 直接影響穿著、無法修復 | 報廢或返工 |
| ⚠️ 重大缺陷 | 明顯可見、影響美觀 | 二等品或返工 |
| ✅ 輕微缺陷 | 細看才見、不影響使用 | 一等品 |
| ✓ 可接受 | 肉眼難見或在邊緣 | 正常接受 |

### 常見缺陷嚴重度

```
🔴 致命級
├─ 強力不足（易斷）
├─ 色澤極差（與樣色差ΔE>3）
├─ 大面積刮痕（>5cm²）
└─ 整體泳移

⚠️ 重大級
├─ 明顯色斑、色花
├─ 中等折痕（影響美觀）
├─ 異纖、死棉（可見）
└─ 針路、油污

✅ 輕微級
├─ 輕微色差（ΔE<1）
├─ 小面積瑕疵（<2cm²）
├─ 邊角瑕疵（可裁切）
└─ 輕微毛羽

✓ 可接受
├─ 邊界線瑕疵
├─ 肉眼難見缺陷
└─ 不影響功能的小問題
```

## 品管工作常用短語

| 中文 | 英文 | 說明 |
|------|------|------|
| 外觀檢驗 | Visual inspection | 目視檢查 |
| 逆光查看 | Against light | 背光觀察，便於發現缺陷 |
| 合格 | Pass ✓ | 符合標準 |
| 不合格 | Fail ✗ | 不符合標準 |
| 條件接受 | Conditional accept ⚠️ | 有缺陷但客戶同意 |
| 送測 | Send for testing | 送到實驗室檢測 |
| 返工 | Re-processing | 重新加工修正 |
| 報廢 | Waste | 無法使用，丟棄 |
| 降級 | Downgrade | 從優等品降為一等品 |
| 改善 | Improvement | 工藝改進 |
