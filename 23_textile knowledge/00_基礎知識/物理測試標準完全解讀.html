色牢度、縮率、強度 — 國際測試方法與標準解析

💪 A. 強度相關測試

1. 斷裂強度

定義

纖維/紗線/織物在拉伸至斷裂時的最大力

單位

N（牛頓）、cN（厘牛頓）

意義

直接反映材料抗撕裂能力。數值越大，抗撕裂性越強。

應用場景

運動服、工作服、承重製品檢驗

2. 撕裂強度

定義

在預先割口的織物上施加拉力，使其繼續撕裂時的力

與斷裂強度的區別

✓ 斷裂強度：從完整起點拉扯至斷
✓ 撕裂強度：從已有割口繼續撕裂

評估意義

評估已有損傷時的承力能力

3. 接縫強度

定義

縫合部分在拉力下的最大抗力

為什麼重要

縫合通常是成衣最薄弱的環節，決定成衣實際承力

應用

成衣品質檢驗的關鍵指標

4. 彈性恢復率

定義

拉伸後返回原長度的比例

彈性恢復率(%) = (原長度 - 拉伸後長度) / 原長度 × 100%

評估對象

織物彈性、鬆緊度、耐穿性

5. 初始拉伸

定義

開始拉伸時的柔性水平

分級

低 / 中 / 高

穿著影響

直接影響穿著舒適度。初始拉伸越小，越易有頂感。

📏 B. 尺寸穩定性測試

縮率測量公式

縮率(%) = (原始尺寸 - 測試後尺寸) / 原始尺寸 × 100%

例如：原始 100cm，測試後 97cm
縮率 = (100-97)/100 × 100% = 3%
測試類型 定義 測試方法 容許值 應用
水洗縮率 水洗後尺寸變化 ISO 6330 規定溫度/次數 通常 ±3% 日常穿著預測
乾洗縮率 乾洗後尺寸變化 乾洗溶劑標準 通常 ±3% 高級成衣
汗液縮率 人工汗液浸泡後變化 ISO 3160-2 較寬鬆，±5% 運動服檢驗
烘乾縮率 高溫烘乾後收縮 時間/溫度規定 根據纖維類型 快乾特性評估

縮率控制技巧

降低縮率的方法：
✓ 前處理充分（去脂、去漿）
✓ 採用防縮整理
✓ 烘乾溫度不宜過高
✓ 含化纖比例提高
✓ 織造密度合理設計

👀 C. 外觀與手感評估

項目 定義 評估方式 標準參考
起毬（Pilling） 織物表面形成小球體 日本標準 (JIS) 1-5級，5級最好
鉤絲（Snag） 織物表面被鉤出絲線 目測+觸摸檢查 越少越好
褪色/色牢度 顏色在光、洗、摩擦下的穩定性 ISO 105 標準系列 3-5級，5級最高
懸垂性 織物自然下垂的柔和度 懸垂係數計算 60-80%為優
光澤度 織物表面光線反射程度 儀器測定 根據設計要求
硬挺度 織物的抗彎剛度 力學測定 根據用途定

💨 D. 透氣/透濕/防水性測試

透氣量（Air Permeability）

單位： cm³/(s·cm²)

含義： 單位時間內通過織物的空氣量

應用： 決定織物「呼吸性」

更高 = 更透氣

透濕量（Moisture Vapor Transmission）

單位： g/(m²·24h)

含義： 水蒸氣通過織物的速率

應用： 舒適性指標

更高 = 更舒適

撥水性（Water Repellency）

單位： 等級（JIS/ISO）

含義： 織物抵抗水分滲入的能力

應用： 防雨性能

更高 = 更防雨

防水性（Waterproofness）

單位： cm H₂O（靜水壓）

含義： 織物被穿透時所需的水柱高度

應用： 防水強度量化

更高 = 防水更強

防水等級參考

靜水壓（cm H₂O） 防水級別 應用場景
< 50 撥水 日常防汗
50-500 防水 一般防雨
500-10,000 高防水 戶外運動
> 10,000 專業防水 水上作業

🚀 快速參考

❓ 我不知道該測什麼？

按成品用途選擇：

  • 日常穿著：水洗縮率、色牢度、強度
  • 運動服：撕裂強度、汗液縮率、汗液牢度
  • 戶外服：防水性、透濕性、日曬牢度
  • 兒童服：強度、色牢度、縮率、皮膚友善
  • 高級服：全套測試 + 彈性恢復率

💡 常見問題

Q：色牢度 3 級能接受嗎？
A：取決於客戶要求。一般商品 3 級可接受，高級商品需 4-5 級。

Q：縮率 ±5% 是否超標？
A：通常 ±3% 為標準。±5% 需視客戶規範而定。

Q：防水靜水壓多少才算防水？
A：≥50 cm H₂O 算撥水，≥1000 cm H₂O 才算真防水。
