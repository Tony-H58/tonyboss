# 染色疵點診斷與解決方案

## 快速診斷流程

```
發現染色疵點
    ↓
觀察特徵（位置、形狀、顏色）
    ↓
確定疵點類型（見下表）
    ↓
查找原因（工藝/設備/材料）
    ↓
實施解決方案
    ↓
調試與驗證
```

## 13大染色疵點診斷表

### 1. 染色不勻 ⚠️ 中高優先級

| 方面 | 內容 |
|------|------|
| **現象** | 布面顏色深淺不均，出現色斑或色塊 |
| **檢驗方法** | 平鋪布面，逆光觀察，與標樣對比 |
| **主要原因** | 1. 浸染固色過早<br/>2. 分散/活性染料分布不勻<br/>3. 活性長車染色毛效不好 |

**解決方案**

| 原因 | 改善措施 | 優先度 |
|------|---------|-------|
| 浸染固色太早 | 延遲固色時間，讓染料充分擴散 | 🔴 高 |
| 分散/活性分布不勻 | 加強迴圈，檢查泵的流速 | 🔴 高 |
| 毛效不好（長車） | 增加毛效次數，檢查毛輥磨損 | ⚠️ 中 |

### 2. 泳移 🔴 高優先級

| 方面 | 內容 |
|------|------|
| **現象** | 軋染：正反面顏色有深淺<br/>浸染：整體退色，邊緣色深 |
| **檢驗方法** | 對折觀察正反面色差，水洗後對比 |
| **危害** | 影響最終顏色，難以補救 |

**浸染泳移的原因與方案**

| 原因 | 解決方案 | 驗證方法 |
|------|---------|---------|
| 染料牢度差 | 更換高牢度染料<br/>選擇適合的活性/還原染料 | 對比小樣 |
| 紗線洗滌不夠 | 增加前處理洗滌次數<br/>確保紗線去淨表面污垢 | 檢驗紗線白度 |
| 未經固色 | 按工藝標準執行固色<br/>確保純鹼和元明粉用量 | 檢查投料記錄 |
| 紗線含鹼太多 | 酸洗調節pH至5.5-6.5 | pH試紙檢驗 |
| 長期乾燥環境 | 適當加濕，控制儲存環境 | 濕度計監控 |
| 混合纖維沾汙 | 分層浸液，防止纖維間接觸 | 工藝改進 |

**軋染泳移的方案**

| 改善措施 | 目的 |
|---------|------|
| 增加軋液壓力 | 加快染料滲透速度 |
| 減少帶液率 | 降低表面液量，均衡染料分布 |
| 增加毛效 | 使染料儘快均勻擴散到織物內 |
| 選小顆粒分散染料 | 提高擴散速率 |
| 控制車速 | 給予布足夠接觸時間 |
| 紅外線預烘 | 達到預期效果，穩定顏色 |

### 3. 色斑（色點） ⚠️ 中高優先級

| 方面 | 內容 |
|------|------|
| **現象** | 布面出現深色點狀或塊狀，顏色深於正常布面 |
| **檢驗方法** | 目視檢查，逆光查看，與標樣對比 |
| **常見位置** | 邊角、褶皺處 |

**原因與解決**

| 原因 | 具體情況 | 改善方案 |
|------|---------|---------|
| **加鹽/加鹼未溶解** | 浸染加鹽、加鹼後未及時迴圈稀釋 | ✅ 先溶解後加入<br/>✅ 加入後立即迴圈10分鐘 |
| **染料未完全溶解** | 分散染料沒有磨碎<br/>難溶染料未過濾 | ✅ 分散染料要磨碎<br/>✅ 難溶染料加熱溶解後過濾<br/>✅ 加熱後再加入染缸 |
| **固色劑提前加入** | 高溫時提前加純鹼/元明粉 | ✅ 按溫度節點加入<br/>✅ 60°時加色素<br/>✅ 冷至50°時加鹽<br/>✅ 50°時加鹼固色 |

### 4. 色條 ⚠️ 中優先級

| 方面 | 內容 |
|------|------|
| **現象** | 布面呈現規律性的縱向或橫向條紋 |
| **多發布種** | 高密府綢（對工藝要求高） |
| **根本原因** | 前處理、染色設備要求高，極易起皺 |

**改善方案**

| 工序 | 改善措施 |
|------|---------|
| **前處理** | ✅ 加強燒毛均勻性<br/>✅ 煮布時避免成繩狀<br/>✅ 漂白時確保液體流暢 |
| **染色** | ✅ 使用適合布種的染色機<br/>✅ 確保布料在缸內流暢<br/>✅ 增加迴圈次數 |
| **機械** | ✅ 檢查鼓輪磨損<br/>✅ 確保軸向受力均勻 |

### 5. 筒子（經軸）紗內外色差 ⚠️ 中高優先級

| 方面 | 內容 |
|------|------|
| **現象** | 卷筒紗中心（內）與表面（外）顏色深淺不同 |
| **檢驗方法** | 沿軸向切面檢查顏色層次 |
| **影響** | 織造時顏色不穩定 |

**診斷與解決**

| 原因 | 具體表現 | 解決方案 |
|------|---------|---------|
| **染料選擇不合理** | 上色速率快慢不一致 | ✅ 選擇上色速率適中的染料<br/>✅ 通過加鹽、加鹼次數與數量調整<br/>✅ 控制迴圈時間（內←→外） |
| **加鹽/加鹼過量** | 第一次加入量太多或未溶解 | ✅ 減少第一次加入量<br/>✅ 必須先溶解稀釋<br/>✅ 分次加入（第1/2/3次） |
| **升溫速率不合理** | 升溫太快或溫度設定過高 | ✅ 選擇適合染料的升溫速率<br/>✅ 不同染料升溫速度不同：<br/>活性：2-3°C/min<br/>分散：3-5°C/min<br/>✅ 最終溫度要符合染料性質 |

### 6. 缸差 🔴 高優先級

| 方面 | 內容 |
|------|------|
| **現象** | 同一次染色中不同批次的缸顏色不同 |
| **影響** | 整批訂單色系不一，無法交貨 |
| **嚴重性** | 最難控制的問題之一 |

**根本原因分析**

| 大類 | 具體原因 | 預防措施 |
|------|---------|---------|
| **染料因素** | 不同缸的溫度不同<br/>各工序升溫速度不同 | ✅ 配備溫度表<br/>✅ 標準化升溫曲線<br/>✅ 定期校準溫度 |
| **配方因素** | 染料組合不好<br/>染料穩定性差 | ✅ 小樣確認無缸差<br/>✅ 使用穩定性好的染料<br/>✅ 避免易變色組合 |
| **操作因素** | 操作不穩定<br/>人員差異 | ✅ 標準化操作流程<br/>✅ 人員培訓與檢查<br/>✅ 記錄每個步驟 |
| **材料因素** | 染料含潮率變化<br/>化解溫度不當 | ✅ 新舊染料分開<br/>✅ 定期檢驗含潮率<br/>✅ 按染料說明書化解 |
| **設備因素** | 盛裝染料的工具不乾淨 | ✅ 專用工具專用杯<br/>✅ 使用前清洗乾淨<br/>✅ 定期消毒 |

**預防方案（優先順序）**

1. 🔴 小樣驗證（先做小樣，確認無缸差）
2. 🔴 溫度控制（多缸需同步監控）
3. ⚠️ 操作標準化（SOP培訓）
4. ⚠️ 原料穩定（定期檢驗）
5. ✅ 設備維護（定期校驗）

### 7. 色光不對 ⚠️ 中優先級

| 方面 | 內容 |
|-----|------|
| **現象** | 成品顏色與標樣不符 |
| **差異類型** | 偏紅/偏藍/偏黃/偏綠等 |
| **影響** | 客戶退貨，信譽損失 |

**多因素分析**

| 原因類別 | 具體原因 | 改善措施 |
|---------|---------|---------|
| **配方因素** | 三原色用量接近<br/>顏色對光線敏感<br/>對溫度敏感 | ✅ 調整色光偏向<br/>✅ 小樣多次驗證<br/>✅ 選擇穩定性好的組合 |
| **工藝因素** | 染色/後整理溫度偏差<br/>時間偏差<br/>易泳移 | ✅ 嚴格控制溫度±2°C<br/>✅ 嚴格控制時間<br/>✅ 確保工藝執行 |
| **材料因素** | 從小樣到大樣放大不准 | ✅ 小樣升倍3-5次驗證<br/>✅ 分析放大比例<br/>✅ 建立放大公式 |
| **混合纖維** | 不同纖維上色率差異 | ✅ 分別測試各纖維<br/>✅ 按纖維比例調配<br/>✅ 選擇適合混紡的染料 |
| **前處理** | 浮色未洗淨<br/>pH值太高 | ✅ 增加皂洗次數<br/>✅ 檢驗pH值（應在6.5-7.5） |

### 8. 白芯 🔴 高優先級

| 方面 | 內容 |
|------|------|
| **現象** | 紗線中心顏色淡，周邊深，呈現分層現象 |
| **多發** | 厚重布料、高密度紗線 |
| **後果** | 織造後顏色不穩定 |

**原因與解決**

| 原因 | 情況說明 | 解決方案 |
|------|---------|---------|
| **前處理不透** | 燒毛不徹底，纖維層未完全開放 | ✅ 加強燒毛力度<br/>✅ 增加煮布時間<br/>✅ 檢查滲透劑加量 |
| **加鹽/加鹼過快** | 第一次加入量太大，外層馬上上色 | ✅ 減少第一次加入量<br/>✅ 分次加入：<br/>  第1次：加50%<br/>  第2次：加50%<br/>✅ 加入後要迴圈均勻 |

**預防措施**

```
高密度/厚重布
    ↓
前處理特別加強（燒毛+煮布+漂白都要充分）
    ↓
加鹽/加鹼分次加入，不可一次性加滿
    ↓
延長迴圈時間，確保內外滲透
    ↓
檢驗：沿軸向切面查看顏色均勻性
```

### 9. 色淺 ⚠️ 中優先級

| 方面 | 內容 |
|------|------|
| **現象** | 成品顏色比標樣淺 |
| **客戶反應** | 接近但偏淺，無法交貨 |

**原因與快速調整**

| 原因 | 判斷方法 | 調整方案 |
|------|---------|---------|
| **加料不足** | 核對投料單 | ✅ 補加不足的染料<br/>✅ 重新升溫至工藝溫度<br/>✅ 保溫30-60分鐘 |
| **染缸漏料** | 檢查缸體 | ✅ 檢查進液口、出液口<br/>✅ 檢查閥門是否關閉<br/>✅ 停機檢修 |
| **顏色太深紗未改性** | 檢驗紗線強力和性質 | ✅ 先做小樣選擇改性劑<br/>✅ 改性後再染深色<br/>✅ 測試最終牢度 |
| **化解溫度不當** | 檢查加熱溫度 | ✅ 按染料說明書溫度化解<br/>✅ 通常分散130°，活性50-60° |
| **配方不合理** | 分析小樣配方 | ✅ 重新設計配方<br/>✅ 增加主色比例<br/>✅ 小樣驗證後上機 |

### 10. 顏色牢度不好 🔴 高優先級

| 方面 | 內容 |
|------|------|
| **測試項目** | 摩擦牢度、日曬牢度、洗滌牢度 |
| **根本原因** | 主要受染料結構影響 |

**牢度不好的原因與改善**

| 牢度類型 | 原因 | 改善措施 |
|---------|------|---------|
| **摩擦牢度差**<br/>（日常磨損） | 顏色太深，浮色多 | ✅ 減淡顏色<br/>✅ 增加皂洗次數<br/>✅ 使用固色劑 |
| **日曬牢度差**<br/>（長期暴曬） | 顏色太淺，染料本身等級低 | ✅ 選擇高牢度染料<br/>✅ 增加濃度（顏色深一點）<br/>✅ 加助劑改善 |
| **洗滌牢度差**<br/>（水洗） | 浮色未洗淨<br/>pH值太高 | ✅ 增加皂洗次數<br/>✅ 冷水水洗徹底<br/>✅ 控制最終pH至6.5-7.5 |

**助劑改善方案**

| 症狀 | 使用的助劑 |
|------|-----------|
| 摩擦牢度一般 | 陽離子固色劑（陽固） |
| 日曬牢度不足 | 日曬牢度改善劑（UV吸收劑） |
| 洗滌牢度不足 | 阻游劑 + 提高固色劑用量 |

### 11. 高溫沾色 ⚠️ 中高優先級

| 方面 | 內容 |
|------|------|
| **現象** | 高溫環境下（烘乾、熨燙）相鄰布料相互沾色 |
| **多發** | 深色布 |
| **原因** | 昇華牢度差 |

**解決方案**

| 原因 | 改善措施 |
|------|---------|
| **昇華牢度差** | ✅ 選擇昇華牢度好的分散染料<br/>✅ 增加固色溫度（180-220°C）<br/>✅ 適當延長定型時間 |
| **未經固色** | ✅ 按工藝標準執行定型<br/>✅ 確保達到所需溫度 |
| **後整理不充分** | ✅ 增加後整理溫度<br/>✅ 使用固色劑輔助<br/>✅ 定型機溫度要穩定 |
| **pH值太高** | ✅ 檢驗成品pH值<br/>✅ 酸洗調節至6.5-7.5 |

### 12. 紗硬 ⚠️ 中優先級

| 方面 | 內容 |
|------|------|
| **現象** | 紗線手感粗硬，缺乏柔軟性 |
| **影響** | 針織織造困難，成品舒適度差 |

**多因素分析**

| 原因 | 具體情況 | 改善措施 |
|------|---------|---------|
| **原紗本身太硬** | 紗線製造時張力太高 | ✅ 更換紗源<br/>✅ 與紗廠溝通 |
| **水的硬度太高** | 水中鈣、鎂離子太多 | ✅ 使用軟化水<br/>✅ 加入軟水劑 |
| **前處理不足** | 燒毛、煮布效果差 | ✅ 加強前處理<br/>✅ 增加滲透劑<br/>✅ 提高溫度和時間 |
| **加鹼過多** | 煮布或染色時加鹼太多 | ✅ 減少加鹼量<br/>✅ 控制在標準量<br/>✅ 最後要酸洗中和 |
| **柔軟劑效果不好** | 選擇或用量不合理 | ✅ 更換柔軟劑品牌<br/>✅ 增加用量<br/>✅ 確保充分接觸 |
| **用量不夠** | 柔軟劑加量不足 | ✅ 增加柔軟劑用量<br/>✅ 延長處理時間 |

### 13. 紗線強力差 🔴 高優先級

| 方面 | 內容 |
|------|------|
| **現象** | 紗線易斷，織造困難，廢品率高 |
| **表現形式** | 整體脆損 或 點斷現象 |
| **後果** | 報廢，無法使用 |

**原因與預防**

| 原因 | 具體表現 | 改善措施 |
|------|---------|---------|
| **前處理過度** | 強力下降明顯 | ✅ 檢查前處理溫度<br/>✅ 減少鹼洗時間<br/>✅ 避免過度煮布 |
| **水中重金屬** | 局部斷裂 | ✅ 檢驗水質<br/>✅ 更換水源或用軟化水<br/>✅ 定期檢測水質 |
| **加酸過多** | 特別是高溫時加酸 | ✅ 減少酸用量<br/>✅ 冷溫時加酸（50°C以下）<br/>✅ 標準：0.5-1.0 g/L |
| **染後pH太低** | 最終成品偏酸 | ✅ 檢驗最終pH值<br/>✅ 調節至6.5-7.5<br/>✅ 必要時加鹼中和 |

**檢驗與驗收**

```
紗線強力差
    ↓
測試強力值（與標準對比）
    ↓
檢驗水質（pH、硬度、重金屬）
    ↓
檢驗成品pH值
    ↓
追溯前處理記錄
    ↓
確定改善措施並驗證
```

## 標準工藝參數速查

### 活性染料（棉、麻、粘膠）
- **浸染升溫**：2-3°C/min
- **最高溫**：60°C
- **加鹽溫度**：40-50°C
- **加鹼溫度**：50°C
- **保溫時間**：30-60分鐘
- **最終pH**：6.5-7.5

### 分散染料（涤綸）
- **升溫**：3-5°C/min
- **最高溫**：130°C
- **保溫時間**：30-60分鐘
- **化解溫度**：70-80°C（先化解）
- **載體用量**：2-3 g/L（無高溫機時）

### 還原染料（深色）
- **浸軋溫度**：常溫
- **汽蒸溫度**：102°C
- **汽蒸時間**：1分鐘
- **最終pH**：6.5-7.5
