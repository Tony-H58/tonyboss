| 項目 | 自動化腳本 | 手動腳本 | 選擇菜單 |
|------|----------|---------|--------|
| **檔案名** | 週報自動.ps1 / 月報自動.ps1 | 週報_完整.ps1 / 月報_完整.ps1 | 週報.bat / 月報.bat |
| **詢問** | ❌ 無 | ❌ 無 | ✓ 選擇 |
| **暫停** | ❌ 無 | ❌ 無 | 無 |
| **速度** | ⚡⚡⚡ 最快 | ⚡⚡⚡ 最快 | ⚡⚡⚡ 最快 |
| **執行方式** | 直接執行 | 直接執行 | 選擇 1 或 2 |
| **模式** | 固定快速 | 可選 (無選項) | 可選標準/快速 |
| **用途** | 自動化/排程 | 手動一鍵執行 | 手動選擇模式 |
| **窗口** | 自動關閉 | 完成後關閉 | 選擇後執行 |
| **推薦** | ⭐⭐⭐ 最佳 | ⭐⭐⭐ 最佳 | ⭐⭐ 備用 |

## 🚀 三種執行方式詳解

### 1️⃣ 自動化腳本 (完全無人值守)

**檔案:**
```
週報自動.ps1    ← 週報一次性自動化
月報自動.ps1    ← 月報一次性自動化
```

**特點:**
- ✓ 完全自動化，無任何詢問
- ✓ 自動檢測下載，自動監控 (1秒/次)
- ✓ 最大等待 3 分鐘後自動停止
- ✓ 完全採用快速模式 (3-5倍速)
- ✓ 執行完成自動關閉窗口
- ✓ 適合排程自動執行

**執行方式:**
```powershell
# 直接執行
.\週報自動.ps1
.\月報自動.ps1

# 帶參數執行
.\週報自動.ps1 -Fast
.\月報自動.ps1 -Fast
```

**執行流程:**
```
[START]
  ↓
[檢查源檔]
  ├─ 有: 複製 → 轉檔 → 輸出 → [END]
  └─ 無: 打開瀏覽器
      ↓
    [自動監控 1秒檢查]
      ↓
    [檢測到檔案]
      ↓
    [複製 → 轉檔 → 輸出] → [END]
```

**預期時間:**
- 檔案已存在: 20-25 秒
- 需要下載: 80-250 秒 (3-4 分鐘)
- 最差情況: 不超過 5 分鐘 (3 分鐘監控上限)

**最佳使用場景:**
- 📅 Windows 排程自動執行
- 🤖 後台自動化流程
- ⏰ 定時執行 (每週四、每月 5-8 日)
- 🔄 批量連續執行

---

### 2️⃣ 手動腳本 - 直接執行版本

**檔案:**
```
週報_完整.ps1    ← 週報完整流程
月報_完整.ps1    ← 月報完整流程
```

**特點:**
- ✓ 直接執行，無任何詢問
- ✓ 自動檢測下載，自動監控
- ✓ 完全採用快速模式
- ✓ 簡化日誌輸出
- ✓ 完成後自動關閉
- ✓ 適合手動一鍵執行

**執行方式:**
```powershell
# 直接執行
.\週報_完整.ps1
.\月報_完整.ps1

# 帶快速參數
.\週報_完整.ps1 -Fast
.\月報_完整.ps1 -Fast
```

**執行流程:**
```
[START]
  ↓
[計算日期]
  ↓
[檢查源檔]
  ├─ 有: 顯示 "✓ 找到源檔案，開始轉檔"
  └─ 無: 打開下載系統
      ↓
    [提示手動下載]
      ↓
    [自動監控 3秒檢查]
      ↓
    [檢測完成]
      ↓
[複製 → 轉檔 → 輸出] → [END]
```

**預期時間:**
- 檔案已存在: 30-40 秒
- 需要下載: 150-700 秒 (2-11 分鐘)

**最佳使用場景:**
- 👤 手動一鍵執行
- 💻 桌面直接雙擊 (透過 .bat)
- 🎯 快速處理報表
- 📊 即時執行

---

### 3️⃣ 手動腳本 - 選擇菜單版本

**檔案:**
```
週報.bat         ← 週報選擇菜單
月報.bat         ← 月報選擇菜單
```

**特點:**
- ✓ 提供模式選擇 (1=標準, 2=快速)
- ✓ 根據選擇執行對應模式
- ✓ 易於使用的菜單介面
- ✓ 適合初次使用或調試

**執行方式:**
```batch
雙擊: 週報.bat 或 月報.bat

然後選擇:
  1. 標準模式 (詳細日誌)
  2. 快速模式 (簡化日誌)
```

**執行流程:**
```
[START]
  ↓
[顯示菜單]
  ↓
[用戶選擇]
  ├─ 1: 執行標準模式
  │  └─ 詳細日誌輸出
  └─ 2: 執行快速模式
     └─ 進度點輸出
  ↓
[完整流程] → [END]
```

**預期時間:**
- 選擇 + 執行: 標準模式 50-70 秒，快速模式 30-40 秒

**最佳使用場景:**
- 🎓 學習和理解流程
- 🐛 調試和故障排除
- 👁️ 需要詳細日誌
- 📝 追蹤執行細節

## 🎯 如何選擇？

### 使用 **自動化腳本** 當：
```
✓ 需要完全自動執行
✓ 用於 Windows 排程
✓ 無人值守後台運行
✓ 批量連續執行多個報表
✓ 不需要看執行日誌
✓ 希望最快完成 (3-5倍速)
```

**啟動方式:**
```powershell
# PowerShell 直接執行
.\週報自動.ps1

# Windows 排程執行
powershell.exe -NoProfile -ExecutionPolicy Bypass -File "E:\88. Claude\02_download rawdata\週報自動.ps1"
```

---

### 使用 **手動腳本 (直接)** 當：
```
✓ 手動一鍵快速執行
✓ 桌面雙擊啟動
✓ 需要完全自動化
✓ 不需要選擇模式
✓ 想要簡化執行流程
✓ 定期手動執行報表
```

**啟動方式:**
```powershell
# PowerShell 執行
.\週報_完整.ps1
.\月報_完整.ps1

# 透過 .bat 執行 (桌面雙擊)
週報.bat
月報.bat
```

---

### 使用 **手動腳本 (菜單)** 當：
```
✓ 需要選擇執行模式
✓ 想看詳細日誌
✓ 調試或故障排除
✓ 第一次使用
✓ 需要監控執行進度
✓ 學習系統工作流程
```

**啟動方式:**
```batch
雙擊: 週報.bat 或 月報.bat
然後選擇: 1 (標準) 或 2 (快速)
```

## ⚙️ 技術細節

### 自動化腳本的優勢
```powershell
# 參數配置
-Fast    # 快速模式 (默認啟用)

# Excel 優化
ScreenUpdating = $false      # 禁用屏幕更新 (2-3倍快)
Calculation = Manual         # 手動計算 (1-2倍快)

# 下載監控
檢查間隔: 1 秒 (vs 標準 3 秒)
最大等待: 3 分鐘 (vs 標準 10 分鐘)
```

### 手動腳本的特點
```powershell
# 可選配置
-Fast    # 快速模式 (可選)

# 交互式
提示打開瀏覽器: 是
要求手動下載: 是
監控下載: 自動

# 日誌
詳細輸出: 可選
進度顯示: 實時
```
