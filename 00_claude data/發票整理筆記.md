## 輸出檔命名規則
- 資料**累加**方式記錄，依月份順序逐月寫入同一檔案
- **進行中**（全年未完成）：`發票整理記錄_yymmdd.xlsx`（yymmdd = 當天處理日期）
  - 例：`發票整理記錄_260604.xlsx`
- **全年完成後**：重新命名為 `發票整理記錄_yyyy.xlsx`（yyyy = 發票所屬西元年）
  - 例：`發票整理記錄_2025.xlsx`

## 欄位結構（發票明細）
| 欄 | 欄位 |
|----|------|
| A | 賣方名稱 |
| B | 賣方統編 |
| C | 買方名稱（由統編比對 NameList） |
| D | 買方統編 |
| E | 發票號碼 |
| F | 發票日期 |
| G | 交易日期（費用期間） |
| H | 總計（= I + J） |
| I | 應稅銷售額（由圖片讀取） |
| J | 稅額（由圖片讀取） |
| K | Uber費用 未稅金額（= I - 其他品項） |
| L | 廣告費用 未稅金額 |
| M | 附加服務費 未稅金額 |
| N~T | 品項4~10 未稅金額 |

## 計算規則
- 品項未稅 = round(含稅 / 1.05)
- **Uber費用未稅 = I - 其他所有品項未稅（確保 K+L+M = I）**
- H = I + J

## 處理流程
1. ZIP 解壓提取發票圖片
2. 第一次讀圖辨識資料
3. 計算未稅金額（Uber費用用差額確保合計正確）
4. 寫入發票整理記錄
5. **二次讀圖比對驗證**
6. 有差異 → 修正 → 再次比對（直到正確）
7. 套用格式（標題置中粗體、A~G靠左、H~T靠右）
8. 清理暫存圖片
9. **整個 xlsx 所有工作表處理完成** → 移至 `completed rawdata\` 資料夾
