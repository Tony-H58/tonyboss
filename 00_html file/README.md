## 🚀 使用方式

### 快速查看
```
1. 打開 品管儀表板_總頁.html
2. 點擊連結查看詳細儀表板
3. 或直接打開需要的 HTML 檔案
```

### 更新流程

**週報完成後** → 儀表板自動更新:
```
執行 週報自動.ps1 
  ↓
生成新的 insprecord_QF
  ↓
Python 腳本自動生成 HTML
  ↓
品管儀表板_A/B.html 更新
```

**月度分析** → 生成比對報告:
```
執行 月報自動.ps1
  ↓
執行 比對.py
  ↓
生成 比對結果.html
```

**績效評估** → 生成 KPI 報告:
```
執行月度評估
  ↓
執行 qc_kpi_final.py
  ↓
生成 qc_kpi_report.html
```
