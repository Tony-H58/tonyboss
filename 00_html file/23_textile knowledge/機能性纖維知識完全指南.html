🌡️ B. 濕度調節機能

技術原理

親水化輻射接枝聚合技術
用親水性單體 + 能源照射 → 與疏水性高分子纖維接枝聚合
使疏水性纖維本身具備親水基
實現優越的吸放濕 + 濕度調節功能

應用領域

主要應用：

  • 內衣 — 貼身舒適
  • 運動服 — 快速蒸散
  • 寢具 — 保持乾爽

❄️ D. 溫度調節機能

方法 1：熱轉相材料（PCM）

原理

微膠囊內含相變化材料，吸收/釋放潛伏熱

作用機制（3步）

  1. 冷卻效應：吸收人體多餘熱量（溫度升至融點為止）
  2. 絕緣效應：相變化材料放熱形成熱障，防止人體熱能流失
  3. 調節效應：溫度變化 → PCM產生吸熱/放熱 → 體表溫度變化率接近常數

混入方式

  • 纖維製程中直接混入微膠囊
  • 以 PU 為載體，塗佈於織物表面

方法 2：五單糖纖維（Xylifresh）

特徵

品牌 Xylifresh（日本富士紡）
成分 白樺樹、玉蜀黍芯、椰子殼提取的天然糖醇類（五單糖）
原理 五單糖吸收水分時產生吸熱反應 → 涼感效果
效果 可降溫約 1℃
應用 T恤、內衣、涼感產品
