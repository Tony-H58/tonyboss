📌 GU品質體系概覽

GU（日本優衣庫集團）
是聚陽重要的國際客戶，對品質要求極為嚴格。

驗布標準：10點制（A-F 等級）
驗收原則：零缺陷目標，容忍度極低
不合格品：直接退貨或報廢處理

GU等級分佈視覺化

關鍵區別：邊部 vs 中部

邊部（1英寸寬度）：
缺陷扣分 = 正常扣分 ÷ 2
例如：邊部5cm的色差正常扣2點，邊部則扣1點

中部（其餘區域）：
缺陷扣分 = 標準扣分
按缺陷種類和大小判定

🔄 GU四大驗布流程

1️⃣ 頭缸驗布

時間：染色後立即進行
對象：第一缸布料
目的：確保色澤符合標準
標準：必須 A 或 B 等級

2️⃣ 中段驗布

時間：生產過程中每隔2小時
對象：中途產出的布料
目的：及時發現品質變化
標準：B 等級及以上

3️⃣ 夜間驗布

時間：夜班特定時段
對象：夜班產出的全部布料
目的：確保24小時品質穩定
標準：C 等級以上

4️⃣ 差異核實

時間：發現品質問題時
對象：等級判定有分歧的布料
目的：確保評分公正一致
標準：複審決定最終等級

驗布流程細節

📋 逐碼掃描檢驗

  1. 每碼布料在標準光源下完整掃描
  2. 記錄每碼的缺陷類型、位置、數量
  3. 計算該碼的總扣點
  4. 根據扣點判定該碼的等級（A-F）
  5. 將該碼布料按等級歸類

🎯 等級判定標準

  1. A 等級：0 點，基本無缺陷
  2. B 等級：1-2 點，微小缺陷
  3. C 等級：3-4 點，輕微缺陷
  4. D 等級：5-6 點，明顯缺陷
  5. E 等級：7-8 點，嚴重缺陷
  6. F 等級：9-10 點，廢品

⭐ GU五大核心要求

1. 色澤精度
✓ 必須與標準樣完全匹配
✓ ΔE ≤ 0.5（遠低於業界 0.8）
✓ 整批布料色澤必須一致
2. 無缺陷目標
✓ A 級布料比率 ≥ 85%
✓ B 級布料 ≤ 10%
✓ C 級以下 ≤ 5%
3. 規格精度
✓ 幅寬 ±0.5%（非常嚴格）
✓ 克重 ±3%
✓ 長度 ±1%
4. 物理性能
✓ 色牢度 4 級及以上
✓ 縮率 ±2%
✓ 強力符合規範
5. 文件完整
✓ 每批必須有驗布報告
✓ 樣品留存 1 年以上
✓ 問題追溯記錄完整

❓ GU驗布常見問題

Q：同一缺陷，中部和邊部扣點怎麼計算？
A：邊部（1英寸寬）扣點 = 中部扣點 ÷ 2。例如中部色差扣4點，邊部則扣2點。

Q：C級布料能發給GU嗎？
A：C級只能作為特殊處理布料（降價銷售或修復後銷售），通常不直接發貨GU。

Q：F級布料怎麼處理？
A：F級直接報廢，不可銷售。GU對不合格品的要求極為嚴格。

Q：色澤0.5的ΔE是什麼意思？
A：ΔE是色差值，越小越接近標準樣。GU要求 ≤0.5，業界通常 ≤0.8。

🚀 GU驗布快速查詢

📝 我是檢驗員，如何快速判定等級？

  1. 逐碼掃描布料，記錄所有缺陷
  2. 對照GU缺陷扣分表，計算總扣點
  3. 留意邊部是否有缺陷（扣點÷2）
  4. 根據總扣點判定等級（0=A、1-2=B、3-4=C...）
  5. 填寫檢驗報告，蓋章確認

🎯 我是主管，如何確保符合GU要求？

  1. 確保A級布料比率 ≥ 85%
  2. 每批必須有驗布報告和樣品
  3. 定期抽檢複驗，確保判定一致
  4. 建立改善機制，針對常見缺陷改善
  5. 與GU保持溝通，及時反饋要求變更
