📌 我要做什麼布料？

❌ 如果做 GU：
必須完美 → A級 ≥ 85% → ΔE ≤ 0.5 → 四驗流程 → 完整追溯

✅ 如果做 GAP/H&M：
標準品質 → 一等品 ≥ 95% → ΔE ≤ 0.8 → 認證檢驗 → 環保文件

💰 如果做 TGT：
性價比優 → 合格品 ≥ 90% → ΔE ≤ 1.0 → 常規檢驗 → 基本文件
💡 實用建議：
✓ 優先改善 GU 品質，其他客戶自然達標
✓ 建立分級檢驗體系，同時服務多個客戶
✓ 與客戶溝通澄清要求，避免誤解
✓ 定期送樣確認色澤，確保準確度
✓ 完整保存文件和樣品，應對客訴

❓ 國際客戶常見問題

Q：同一布料能同時供應GU和TGT嗎？
A：可以，但要分級。GU級的自然符合TGT，但TGT級可能不符合GU要求。

Q：ΔE是什麼？怎麼測？
A：色差值，用分光光度儀測，越小越接近標準。GU最嚴(0.5)，TGT最寬鬆(1.0)。

Q：客戶臨時變更品質標準，怎麼辦？
A：立即與客戶確認書面要求，通知工廠調整工藝，取樣確認。

Q：缺少某個認證（SEDEX/OEKO-TEX），能發貨嗎？
A：否，認證是強制要求。必須完成審廠或認證後才能發貨。
